# SHOP-GYRO

Страница магазина гироскутеров

# Стек технологий

- HTML5: семантическая структура страницы магазина гироскутеров
- CSS: стилизация, с использованием flex, пвсевдоклассов. Также попытка в адаптивность.
- JS: функции сайта, т.е. логика изменений по клику некоторых блоков
